Скрипт add_librosa содержит две функции для извлечения content-based характеристик композиций при помощи библиотеки librosa. Параметр функций path -- местоположение композиции. Вторая функция вычисляет много характеристик -- на выходе часто большие массивы. Первая функция облегчённая, считает несколько характеристик и дополнительно уменьшает размерность массивов, синхронизируя по битам (темпу). Результат работы первой процедуры записывается в json-файл.
