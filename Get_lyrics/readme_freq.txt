Скрипты lyrics_artist.py и lyrics_genre.py составляют список слов по всем текстам песен исполнителя или жанра. По полученному списку составляется словарь, ключами которого являются слова, а значениями -- количество вхождений слова. В файле ipynb пример работы, помимо этого в директории есть сгенерированные на основе словарей файлы. Скрипты для составления списка слов требуют доработки -- порой извлекаются ненужные символы, сгенерированные файлы подвергались ручной доработке.
