Скрипт get_lyrics содержит функции для сбора текстов с сайтов lyrics.wikia.com, metrolyrics.com и azlyrics.com. На вход функциям подаётся название трека и исполнителя и задержка между запросами в параметре sleeping. Скрипт add_lyrics добавляет в id3-теги json-файла текст песни, если его не было ранее. Скрипт get_lyrics_by_artist собирает все тексты одного исполнителя в представленном наборе данных в один список, в котором подсчитывается число вхождений слова, полученный словарь выводится в файл с названием, соответствующим исполнителю. Скрипт test_lyrics проверяет корректность скрипта для сбора текста с сайта двумя способами: diff и сравнение словарей с частотами слов.

Скрипт lyrics_to_file создаёт tsv-файл (на основе json-файла) из двух переменных: name вида artist_nams:title и lyrics, которая является обработанным текстом. Текст обработан следующим образом: убраны знаки препинания, все буквы маленькие, убраны стоп-слова. Полученный tsv-файл подаётся на вход скрипту bow, который создаёт bag-of-words и находит ближайшую по тексту композицию к первому наблюдению файла (для примера). Для наглядности результата bow перенесён в Jupyter Notebook файл bow_example.iypnb.
