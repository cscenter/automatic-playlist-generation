Скрипт get_lyrics содержит функции для сбора текстов с сайтов lyrics.wikia.com и azlyrics.com, поиск на последнем в данный момент не работает (возможно, локальный бан с того времени, когда не было задержки между запросами), поэтому вызовов далее нет. На вход функциям подаётся название трека и исполнителя. Скрипт add_lyrics добавляет в id3-теги json-файла текст песни, если его не было ранее. Скрипт get_lyrics_by_artist собирает все тексты одного исполнителя в представленном наборе данных в один список, в котором подсчитывается число вхождений слова, полученный словарь выводится в файл с названием, соответствующим исполнителю. Скрипт test_lyrics проверяет корректность скрипта для сбора текста с сайта двумя способами: diff и сравнение словарей с частотами слов.
