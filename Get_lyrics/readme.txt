Скрипт требует наличия установленной библиотеки BeautifulSoup не ниже 4.4.0 http://www.crummy.com/software/BeautifulSoup/bs4/doc/ . По заранее извлечённым id3-тегам title и artist из json-файла составляется запрос для поиска текста песни на сайте lyrics.wikia.com. При отсутствии текста происходит поиск на сайте www.azlyrics.com. Отсутствие текста определяется по началу строки (которая получается при извлечении текста) "T [[". После этих символов идёт название автора и песни.
